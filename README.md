# TalonsMathHub

V2 is a multi-page front-end math learning website concept.

## Pages
- `v2/index.html` — landing page
- `v2/activities.html` — interactive activities
- `v2/practice.html` — practice overview
- `v2/about.html` — about page
- `v2/secret.html` — hidden bonus challenge room

## Hidden challenge trigger
On the home page, type `133767` from your keyboard to open the secret challenge page.

## Run locally
```bash
cd v2
python3 -m http.server 4173
```
Then open: http://localhost:4173
